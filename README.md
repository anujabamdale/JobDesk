# Job Desk Application

## Overview

The Job Desk Application is an open-source project designed to streamline the job search and management process. This application allows users to track job applications, manage deadlines, and schedule interviews efficiently.

## Features

- **Job Tracking**: Keep track of all job applications in one place.
- **Deadline Management**: Set and manage application deadlines.
- **Interview Scheduling**: Schedule and manage interviews.
- **API Integration**: Fetch job postings from external sources.
- **Responsive Design**: User-friendly interface compatible with various devices.

## Technologies Used

- **Backend**: Python, Flask
- **Database**: SQLite
- **Frontend**: HTML, CSS, JavaScript
- **APIs**: Integrated RESTful APIs for job listings
- **Version Control**: Git
